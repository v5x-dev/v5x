---
title: "Inspect device state"
description: "Read brain, controller, radio, and smart-port state."
---

`V5SerialDevice` refreshes its state views every 200 milliseconds by default.

```ts
const refreshed = await device.refresh();
if (refreshed.isErr()) throw refreshed.error;

console.log({
  brainAvailable: device.brain.isAvailable,
  battery: device.brain.battery.batteryPercent,
  systemVersion: device.brain.systemVersion.toUserString(),
  fieldConnected: device.isFieldControllerConnected,
  matchMode: device.matchMode,
});

for (const smartDevice of device.devices) {
  console.log(smartDevice.port, smartDevice.type, smartDevice.version);
}
```

Disable `autoRefresh` if your app controls polling itself. Call `refresh()` before reading views that must reflect the latest response.
