title: "Manage programs"

---

## List programs

```ts
const programs = await device.brain.listProgram();
if (programs.isErr()) throw programs.error;

for (const program of programs.value) {
  console.log(program.name, program.slot, program.size);
}
```

## Upload a program

```ts
import { ProgramIniConfig } from "@v5x/serial";

const config = new ProgramIniConfig();
config.baseName = "slot_1";
config.program.name = "Robot";
config.program.slot = 0;
config.autorun = true;

const binary = new Uint8Array(
  await fetch("/robot.bin").then((res) => res.arrayBuffer()),
);
const uploaded = await device.brain.uploadProgram(
  config,
  binary,
  undefined,
  (state, current, total) => console.log(state, current, total),
);

if (uploaded.isErr()) throw uploaded.error;
if (!uploaded.value) throw new Error("Upload was not acknowledged");
```

`ProgramIniConfig.program.slot` is zero-based. The first brain slot is `0` in this metadata object.

## Start and stop a program

Use `setActiveProgram` to load a slot or stop the running program. It
returns `Ok(undefined)` on acknowledgement or `Err(VexSerialError)` on NACK,
timeout, or when the connection isn't open.

```ts
const started = await device.brain.setActiveProgram(1);
if (started.isErr()) throw started.error;

const stopped = await device.brain.setActiveProgram(0);
if (stopped.isErr()) throw stopped.error;
```

The `device.brain.activeProgram` setter is deprecated and will be removed
in a future breaking release. Use `setActiveProgram` instead.
