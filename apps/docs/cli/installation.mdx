## Install with Bun

```bash
bun add --global @v5x/cli
v5x --version
```

The CLI requires Bun 1.3.14 or newer and Bun must be available on your shell's
`PATH`.

The current CLI release supports Linux and macOS, matching the package
metadata's `os` field. Windows is not published yet because the CLI needs a
compatible serial backend there. The separate `@v5x/serial` browser library is
available wherever the browser implements the Web Serial API; browser support
is independent of CLI operating-system support.

## Install a toolchain

<Tabs>
  <Tab title="vexide">
    ```bash
    v5x install vexide
    ```

    This requires Rust and Cargo. It installs `cargo-v5` with Cargo.

  </Tab>
  <Tab title="PROS">
    ```bash
    v5x install pros
    ```

    This requires Python. It installs `pros-cli` into the current user's Python environment.

  </Tab>
</Tabs>

VEXcode C++ projects use the compiler and Make environment supplied by their project setup.
