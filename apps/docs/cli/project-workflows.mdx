---
title: "Project workflows"
description: "Common create, build, upload, and run workflows."
---

## Create a project

```bash
v5x new robot --type vexide
cd robot
```

Use `init` instead when the target directory already exists and is empty.

## Build separately

```bash
v5x build
v5x upload --no-build --file ./path/to/program.bin
```

## Build and run

```bash
v5x run --slot 3 --name "Skills"
```

`run` uses the same upload pipeline as `upload`, then tells the brain to start the program. Slots are numbered 1 through 8.

<Warning>
  Uploading replaces whatever program is in the selected slot. Double-check the
  slot number first.
</Warning>
