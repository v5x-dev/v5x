---
title: "clean"
description: "Remove outputs produced by a supported project toolchain."
---

```bash
v5x clean [path]
```

Alias: `v5x cl`

<ParamField path="path" type="string">
  Project directory. Defaults to the current directory.
</ParamField>

v5x detects the project type and hands cleanup to its toolchain; it fails if it can't detect the project. No brain connection needed.
