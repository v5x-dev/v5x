---
title: "screenshot"
description: "Capture and display the V5 brain screen."
---

```bash
v5x screenshot
```

Alias: `v5x sc`

The brain returns a 480×272 RGB image. v5x writes it to the terminal using the Kitty graphics protocol.

## Save to a file

```bash
v5x screenshot --output screen.png
v5x screenshot --output screen.ppm --format ppm
```

`--output` writes a portable image file instead of using terminal graphics. The default file format is PNG. Use `--format ppm` to write a binary PPM image.

<Note>
  Terminals without Kitty graphics support won't show the image. Use `--output`
  there.
</Note>
