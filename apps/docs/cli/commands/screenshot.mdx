---
title: "screenshot"
description: "Capture and display the V5 brain screen."
---

```bash
v5x screenshot
```

Alias: `v5x sc`

The brain returns a 480×272 RGB image. v5x writes it to the terminal using the Kitty graphics protocol.

<Note>
  The command may produce no visible image in terminals that do not implement
  the Kitty graphics protocol.
</Note>
