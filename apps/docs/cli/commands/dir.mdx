---
title: "dir"
description: "List files stored in the brain's flash vendors."
---

```bash
v5x dir
```

Alias: `v5x ls`

The table includes name, size, load address, timestamp, and CRC32. v5x searches the user, system, VEX, VEX VM, PROS, and development vendor areas.

Displayed paths use prefixes such as `user/`, `sys_/`, `pros/`, `vex_/`, and `vxvm/` to identify the file vendor.
