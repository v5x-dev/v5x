---
title: "init"
description: "Create a project in an existing empty directory."
---

```bash
v5x init [path] --type <pros|vexide>
```

<ParamField path="path" type="string">
  Empty destination directory. Defaults to the current directory.
</ParamField>

<ParamField query="type" type="string" required>
  Project template. Must be `pros` or `vexide`.
</ParamField>

The directory's basename becomes the project name.
