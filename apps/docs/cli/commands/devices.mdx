---
title: "devices"
description: "List smart devices connected to a V5 brain."
---

```bash
v5x devices
```

Alias: `v5x lsdev`

The result contains each smart-port number, detected device type, and firmware version. A connected brain is required.

Device types include motors, controllers, inertial, distance, optical, GPS, vision, ADI, radio, and other devices recognized by the protocol.
