v5x is a set of tools for developing on the VEX V5 platform. It talks directly to a V5 brain over USB from terminal workflows, browser applications, and framework apps.

<CardGroup cols={3}>
  <Card title="Use the CLI" icon="terminal" href="/cli/overview">
    Build, upload, run, and inspect V5 programs from a terminal.
  </Card>
  <Card title="Use the serial library" icon="usb" href="/serial/overview">
    Build browser tools on top of the V5 serial protocol.
  </Card>
  <Card title="Use the web layer" icon="monitor" href="/web/overview">
    Manage browser connection state directly or through React, Svelte, and Solid
    bindings.
  </Card>
</CardGroup>

## Packages

| Package       | Use it when                                                                         |
| ------------- | ----------------------------------------------------------------------------------- |
| `@v5x/cli`    | You want a ready-to-use workflow for PROS, vexide, or VEXcode C++ projects.         |
| `@v5x/serial` | You are building a browser application that connects to a brain through Web Serial. |
| `@v5x/web`    | You want browser connection state plus `react`, `svelte`, or `solid` bindings.      |

<Note>
  v5x is an independent project. It is not an official VEX Robotics product.
</Note>
