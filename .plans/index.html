Global completion criteria

  • bun run typecheck, bun run lint, bun run format:check, bun run test, bun run test:coverage, and bun run build pass.
  • Both published packages pass pack-and-install smoke tests for ESM, CommonJS where supported, declarations, and the CLI binary.
  • No production API launches an unobservable async operation that can reject.
  • Every serial protocol mode transition is serialized and restored in finally.
  • Network-controlled buffers and extracted archives have explicit documented limits.
  • No dev server is required by any verification step.
